# 🌾 Smart Farmer Management & Financial Analysis System

A full-stack web application designed to help farmers efficiently manage their agricultural activities, financial records, and crop data. This project demonstrates practical implementation of Database Management System (DBMS) concepts using real-world scenarios.

---

## 🚀 Features

* 👤 **Farmer Authentication**

  * Secure signup and login system

* 💰 **Expense Management**

  * Add, update, and delete farming expenses
  * Track spending categories (seeds, fertilizers, etc.)

* 💵 **Income Tracking**

  * Record income from crop sales and other sources

* 🌱 **Crop Management**

  * Maintain crop records (sowing, harvesting, yield)

* 📊 **Dashboard & Reports**

  * View total income, expenses, and profit
  * Clean and interactive dashboard UI

* ✏️ **CRUD Operations**

  * Full Create, Read, Update, Delete functionality across all modules

## ⚙️ How to Run Locally

1. Install XAMPP
2. Start **Apache** and **MySQL**
3. Place project folder in:

   ```
   C:\xampp\htdocs\
   ```
4. Import the database using phpMyAdmin
5. Open in browser:

   ```
   http://localhost/farmer-system
   ```
